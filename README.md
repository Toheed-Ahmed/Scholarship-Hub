# 🎓 Scholarship Hunt by AI

**🔗 Live Demo:** [https://scholarship-6637pvsda-myhosts-projects.vercel.app/](https://scholarship-6637pvsda-myhosts-projects.vercel.app/)

**Scholarship Hunt by AI** is a premium, fully responsive, zero-database guidance platform designed to help students worldwide navigate the complex process of securing fully-funded Master's and PhD scholarships. 

Built with modern SaaS aesthetics and Human-Computer Interaction (HCI) best practices, this platform serves as an interactive, step-by-step roadmap offering integrated AI prompts, application tracking, and automated document templates.

---

## ✨ Key Features

### 1. The ScholarAI Framework (Multi-Step Form)
An interactive profiling wizard where students input their academic details, target countries, and document readiness. Progress automatically persists in the browser via `LocalStorage`.

### 2. Embedded AI Prompt Library
A curated library of strategic prompts for ChatGPT, Claude, and Gemini. Users can click to instantly copy prompts for tasks like:
* Scholarship Discovery
* Statement of Purpose (SOP) Evaluation
* Cold Emailing Professors
* Mock Interview Simulations
* CV/Resume Optimization

### 3. Static AI Chatbot
An integrated, lightweight chatbot built natively in JavaScript (no API required) providing immediate, keyword-based advice on common scholarship queries (e.g., CVs, LORs, IELTS).

### 4. Application Tracker Dashboard
A complete CRUD dashboard allowing students to log, monitor, and delete active scholarship applications. The UI switches between a wide data-table on Desktops and stacked cards on Mobile devices. Status updates dynamically sync with the global progress bar.

### 5. Essential Resources & AI Tools Hub
Curated quick-links to top global scholarship portals (DAAD, Chevening, Fulbright, Erasmus+) alongside essential AI writing tools (Grammarly, Hemingway, Resume.io).

### 6. Curated PDF Templates
Access a categorized library of professionally formatted `.pdf` templates for CVs, SOPs, Recommendation Letters, and English Guides. Neatly organized for instantaneous, direct, cross-platform downloads.

## 💻 Tech Stack
* **HTML5** - Semantic structure and accessibility.
* **Vanilla CSS3** - Custom CSS Grid/Flexbox architecture, CSS Variables, Glassmorphism UI, Intersection Observers, and strict Mobile-First Breakpoints (Zero UI frameworks).
* **Vanilla JavaScript (ES6)** - Handles DOM manipulation, LocalStorage sync, intersection scroll animations, and Chatbot logic.

---

## 📱 Mobile-First & Responsive
The entire user interface is governed by strict viewport rules ensuring 100% responsiveness:
* **Desktop (1024px+)**: Flowing 3-column grids, wide tracker tables.
* **Tablet (768px)**: Optimized 2-column breakdowns.
* **Mobile (320px - 480px)**: Stacked single columns, card-based table conversions, fixed-height chat containers, horizontal overflow kill-switches, and anti-zoom typography locking for iOS/Safari.

## 🚀 How to Run Locally
Because this project is entirely static (Client-Side), there is no complex build step required.

2. Open the `index.html` file in any modern web browser.
3. Done!
